Where I am spending the most and why it might be a problem 2. Which specific expenses I can cut down 3. Practical alternative suggestions to reduce costs 4. One key habit I should change Keep it conversational, specific, and actionable. Format with clear sections.""" } ] ) analysis = message.content[0].text except Exception as e: analysis = "AI insights are currently unavailable. Please check your API credits and try again." return render_template('insights.html', analysis=analysis, total=total)
